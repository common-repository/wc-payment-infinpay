== Description ==

infinpay's latest, most complete payment processing solution. Accept credit/debit cards and local payment methods. Turn on only infinpay options or process a full suite of payment methods.
Built and supported by [WooCommerce](https://woocommerce.com) and [infinpay](https://www.infinpay.com/).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woocommerce-infinpay.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Setup and Configuration =

Follow the steps below to connect the plugin to your infinpay account:

1. After you have activated the WooCommerce infinpay Payments plugin, go to **WooCommerce  > Settings**.
2. Click the **Payments** tab.
3. The Payment methods list will include infinpay  options. Click on **infinpay** 
4. Fill in the Merchant Code, API Key, API Secret value which provided from infinpay upon your registration.
5. You may configure the Title, Description for your infinpay
6. Environment: Sandbox or Live allowed you to switch the environment to do testing on Sandbox before your page go to live.
